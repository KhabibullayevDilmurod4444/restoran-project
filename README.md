## 📖 Loyiha haqida

**Milliy Restoran** — bu O'zbekistonning boy va xilma-xil milliy taomlarini namoyish etuvchi zamonaviy veb-ilova. Foydalanuvchilar restoran menyusini qulay tarzda ko'rib chiqishlari, taomlar haqida batafsil ma'lumot olishlari mumkin.

> *"Har bir taom — bu tarix, har bir laqma — bu sevgi."*

---

## ✨ Imkoniyatlar

- 🧆 **Menyu ko'rish** — Barcha milliy taomlar ro'yxati, narxlari va tavsifi
- 🖼 **Taom rasmlari** — Har bir taom uchun chiroyli foto-galereya
- 🔍 **Qidirish** — Taom nomi yoki turi bo'yicha tez qidirish
- 📱 **Responsive dizayn** — Mobil va desktop qurilmalarga moslashgan


## 🤝 Hissa qo'shish

Pull request'lar mamnuniyat bilan qabul qilinadi! Katta o'zgarishlar uchun avval `issue` oching.

1. Forkni oling
2. Yangi branch yarating (`git checkout -b feature/yangi-imkoniyat`)
3. O'zgarishlarni commit qiling (`git commit -m 'Yangi imkoniyat qo'shildi'`)
4. Branch'ga push qiling (`git push origin feature/yangi-imkoniyat`)
5. Pull Request yuboring
